# Smart-Admissions
This project uses data science and technical writing to design a predictive system that forecasts an applicant's likelihood of gaining admission based on key variables like course, department, UTME score, and screening score. It includes visualizations, a prediction model, documentation, and a working prototype dashboard.

# Smart Admissions

This project uses data science and machine learning to predict the success of university admission based on applicant profiles.

## 🔍 Project Objective
To simulate, analyze, and model the relationship between UTME/screening scores, courses, and departments to forecast admission outcomes.

## 🧰 Tools & Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib/Seaborn
- Jupyter Notebook
- Streamlit
- GitHub

## 📁 Project Structure
- `data/`: Simulated datasets
- `notebooks/`: Jupyter notebooks for exploration and modeling
- `scripts/`: Python scripts for data generation and processing
- `visuals/`: Saved plots and charts
- `docs/`: Internal documentation
